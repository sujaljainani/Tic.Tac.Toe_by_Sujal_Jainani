🎮 Tic Tac Toe Game (Python + Flowchart)

📌 Project Overview

This project is a simple 2-player Tic Tac Toe game developed using Python. It demonstrates basic programming concepts such as loops, conditionals, and functions. Along with the code, a RAPTOR flowchart is also included to visually represent the logic of the game.

---

🚀 Features

- 2-player console-based game (X vs O)
- Input validation (prevents invalid moves)
- Win detection logic
- Draw detection
- Simple and clean user interface
- RAPTOR flowchart for better understanding

---

🛠️ Technologies Used

- Python (for game development)
- GitHub (for version control and project hosting)

🔄 Flowchart

The logic of the game is represented using a  flowchart.

- Start the game
- Initialize board
- Take user input
- Validate move
- Check winner
- Check draw
- Switch player
- Repeat until game ends

🧠 Algorithm

1. Initialize an empty board of 9 positions
2. Set current player to X
3. Display the board
4. Take input from player
5. Check if move is valid
6. Place the symbol on board
7. Check for winner
8. Check for draw
9. Switch player
10. Repeat until game ends
